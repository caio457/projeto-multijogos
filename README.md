
# 🎮 Sistema de Jogos

Sistema desenvolvido em linguagem C com o objetivo de simular um ambiente de jogos interativo no terminal.

---

## 🚀 Sobre o projeto

O **Sistema de Jogos 2.0** é uma aplicação em C que reúne diferentes funcionalidades relacionadas a jogos, permitindo interação do usuário através do terminal.

O projeto foi criado com foco em praticar lógica de programação, estrutura de código e organização de sistemas.

---

## ⚙️ Funcionalidades

- 🎯 Menu interativo no terminal
- 🎮 Execução de jogos simples
- 🔄 Navegação entre opções
- 📥 Entrada de dados do usuário
- 📊 Sistema estruturado em funções

---

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Compilador GCC
- Git / GitHub
